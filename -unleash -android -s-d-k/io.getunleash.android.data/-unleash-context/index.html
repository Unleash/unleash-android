UnleashContext

data class UnleashContext(val userId: String? = null, val sessionId: String? = null, val remoteAddress: String? = null, val properties: Map<String, String> = emptyMap())

Allows the consumer to set the properties that will determine whether a flag is on or off for complex strategies and constraints Note that this SDK does not support the current time since this will not work correctly with a proxy SDK. The environment property is no longer exposed, as of v1.0 of this SDK, it's not calculated through the API key. See the documentation
